Flip-Flops

A flip-flop is a device which stores a single bit (binary digit) of data; one of its two states represents a "one" and the other represents a "zero". Such data storage can be used for storage of state, and such a circuit is described as sequential logic in electronics.

    Table 1.2: Excitation Table

    Fig 1.2 depicts SR Flip-Flop which is built using NOR Latch. The flip-flop will function only if E i.e. Enable input is '1'.

    The flip-flop holds its previous state when it's not functioning. Any change in inputs has no effects in output, hence 'X' is don't care condition. Functionality remains same as that of NOR latch. Enable input can be 'level triggered' or 'edge triggered'.
